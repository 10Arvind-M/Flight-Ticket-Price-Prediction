## Flight Ticket Price Prediction  

## Overview  
This assignment is part of MLP (Machine Learning Practice) . The goal of this Kaggle competition is to **predict the price of a flight ticket** based on various input features such as airline, source, destination, duration, stops, and time-related attributes.


## Objective  
Build a machine learning model that accurately predicts the **flight ticket price** based on historical flight data.

---

## Evaluation Metric  
Submissions are evaluated using r2_Score
